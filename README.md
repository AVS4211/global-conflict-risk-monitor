# 🌍 Global Conflict Risk Monitor

An AI-powered dashboard that analyzes global conflict-related news and predicts potential risk levels using **Natural Language Processing (NLP)** and **Time-Series Forecasting**.

This system collects real-time news, performs sentiment analysis, maps geopolitical conflict locations, and forecasts future risk trends using machine learning models.

---

## 🚀 Project Overview

The **Global Conflict Risk Monitor** is designed to analyze geopolitical events by processing global news data and estimating conflict risk.

The system performs the following tasks:

1. Collects real-time news related to global conflicts
2. Performs sentiment analysis on news headlines and descriptions
3. Calculates overall global risk level
4. Stores time-series risk data
5. Predicts future conflict risk using ARIMA forecasting
6. Displays insights through an interactive dashboard

---

## 🧠 Key Features

✔ Real-time news collection using News API  
✔ NLP sentiment analysis using TextBlob  
✔ Risk scoring based on negative news percentage  
✔ Time-series data storage for historical risk tracking  
✔ ARIMA forecasting for predicting future conflict risk  
✔ Interactive Streamlit dashboard  
✔ Global conflict visualization using Geo Map  
✔ Automated dashboard refresh  

---

## 🖥 Dashboard Components

The dashboard displays:

- 📰 **Latest News Updates**
- 😊 **Sentiment Analysis (Positive / Neutral / Negative)**
- ⚠️ **Current Conflict Risk Level**
- 🌍 **Global Conflict Geo Map**
- 📈 **Risk Trend Forecast Graph**
